The purpose of this program is to show how efficient supermarket checkout lanes are depending on
the business, how many lanes are open, and whether express lanes are being utilized. There is a
report discussing the findings.  The program can be compiles and executed using the commands:
javac ShopperSim.java --- java ShopperSim

ShopperSim.java contains all of the attributes for the checkout lanes that can be manipulated
in order to change the business coming in, the number of checkout lanes, and whether express lanes
are to be used.
